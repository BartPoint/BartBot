<h2>If you don't see the Development menu in the menu bar, choose Safari > Preferences, click on Advanced, and then select "Show the Development menu in the menu bar".</h2>
    - Click on the new menu Develop 
    -  Allow Unsigned extension
    - Download BartBot.app 
    - Click on BartBot.app

Now the Popup BartBot will show you "Quit and Open Safari Extensions Preferences".
